# Screenshots

This folder is for real screenshots of the Home Assistant for Garmin builder.

Use screenshots that show the current companion-builder workflow for
GarminHomeAssistant.

Recommended screenshots:

1. `watch-app-settings.png`
   - Shows API URL, Configuration URL, token guidance, and copy buttons.
2. `add-item-entity-picker.png`
   - Shows selecting a Home Assistant entity and auto-filled watch name.
3. `number-picker-setup.png`
   - Shows Number picker behavior with the relevant fields visible.
4. `menu-items-hierarchy.png`
   - Shows top-level items, submenus, nested items, and reorder/edit buttons.
5. `json-preview.png`
   - Shows the generated GarminHomeAssistant JSON preview.
6. `mobile-builder.png`
   - Shows the builder working inside the Home Assistant mobile app.

When adding screenshots, prefer dark-mode Home Assistant screenshots with any
private URLs, tokens, names, or locations blurred or replaced.
